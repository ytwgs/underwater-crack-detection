# underwater-crack-detection
本作品是基于机器视觉的水下土木结构裂纹分割系统，面向跨海大桥、大坝、港口等水下结构安全监测场景，解决水下图像色偏严重、微小裂纹难以识别、人工检测效率低等行业痛点。 系统集成水下图像色彩校正、微小裂纹精细化分割、可视化交互三大核心功能，采用改进型 UNet 网络，结合多尺度特征提取与自适应注意力机制，可精准识别细微裂纹并自动过滤水体、悬浮颗粒等干扰，实现裂纹的快速检测、轮廓分割与尺寸计算。 系统采用 PyQt6 构建可视化界面，操作简单、部署便捷，支持 Windows 运行，可广泛应用于水利工程、海洋工程、桥梁运维等领域，具备较强的工程实用价值与推广前景。
